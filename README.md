# 📊 Netflix Data Analysis using Python

## 🔍 Project Overview

This project analyzes the Netflix dataset to uncover insights about content distribution, genres, ratings, and trends over time.
The analysis includes data cleaning, exploration, and visualization using Python.

---

## 🎯 Objectives

* Clean and preprocess raw dataset
* Analyze Movies vs TV Shows distribution
* Identify Top 10 genres
* Study country-wise content distribution
* Visualize year-wise growth of Netflix content

---

## 🛠️ Tools & Technologies

* Python
* pandas
* matplotlib
* seaborn

---

## 📁 Dataset

* Source: Netflix dataset (CSV file)
* File used: `netflix_titles.csv`

---

## ⚙️ How to Run This Project

### 🔹 Step 1: Download Project

Option 1 (Recommended):

* Click on **Code → Download ZIP** from kaggle
* Extract the ZIP file

---

### 🔹 Step 2: Open Project

* Open the folder in **PyCharm**

---

### 🔹 Step 3: Install Required Libraries

Run this in terminal:

pip install pandas matplotlib seaborn

---

### 🔹 Step 4: Run the Script

Run the file:

python netflix_analysis.py

## 📊 Key Analysis Performed

### ✔ Data Cleaning
